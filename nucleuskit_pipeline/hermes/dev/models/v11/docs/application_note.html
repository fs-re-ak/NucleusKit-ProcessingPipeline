ModelV11 — Facial EMG Emotion Classifier

Application Note  ·  Release: ModelV11_2026-03-21_d  ·  Date: 2026-03-21  ·  RE-AK Trainers

1. Overview

ModelV11 extends V10 with an artefact-rejection gate that runs before the Stage 1 Neutral detector. Windows whose average EMG amplitude (AVG_RMS) exceeds a threshold derived from the Active-class training distribution are immediately returned as Neutral with an artefact flag, preventing glitches and electrical transients from being misclassified as emotional expressions.

All other aspects of the pipeline (signal conditioning, LDA-based Neutral detector, LDA + Mahalanobis KNN emotion ensemble, 5-fold CV evaluation) are identical to V10.

4. Signal Processing & Feature Pipeline

  1. Causal Butterworth bandpass (15.0–45.0 Hz, order 4)
  2. Per-channel RMS over the analysis window
  3. AVG_RMS — mean RMS across all channels (raw, pre-normalisation)
  4. L2 normalisation of the 8 channel RMS values
  5. Feature vector: [HEAD_R_RMS, HEAD_L_RMS, CHEEK_R_RMS, CHEEK_L_RMS, EAR_R_RMS, FOREHEAD_L_RMS, BROW_RMS, NOSE_RMS, AVG_RMS]

5. Classifier Architecture

Artefact Gate V11 NEW

Applied before Stage 1. If AVG_RMS > 14.689611 the window is immediately labelled Neutral (artefact) and both Stage 1 and Stage 2 are skipped. The threshold was derived as the 95.0th percentile of Active-class AVG_RMS in the augmented training set (935 Active samples), multiplied by a safety scale of 1.0.

Stage 1 — Neutral Detector

Shrinkage LDA (solver=eigen, shrinkage=auto). Binary: Neutral vs. Active. Threshold theta=0.5 on the Active posterior.

Stage 2 — Emotion Classifier

Ensemble: Shrinkage LDA (weight 0.6) + Mahalanobis KNN K=7 (weight 0.4).

7. Known Limitations

Artefact gate calibration. The artefact threshold is derived from the Active-class distribution in the training set. Recordings with systematically different signal amplitudes (different headset fit, electrode impedance, or subject) may require re-calibration of the threshold.
Single-subject training data. The model was trained on recordings from one participant. Generalisation to other individuals has not been validated.

8. Quick Integration

import sys
sys.path.insert(0, "interface")

from realtime_classifier import StreamingEMGClassifier

clf = StreamingEMGClassifier(
    clf_dir="classifier",
    window_sec=1.0,
    step_sec=0.5,
    sampling_rate=250,
)

for raw_frame in your_eeg_source:        # (8,) array in μV, 250 Hz
    result = clf.push_sample(raw_frame)
    if result is not None:
        dominant = max(result, key=result.get)
        print(f"{dominant} ({result[dominant]:.2%})")
